Clouds

Types of Clouds

Public Cloud

A public cloud is a multi-tenancy cloud environment in which multiple users share the same resources. A public cloud is separated by multiple security layers from storage to user interface.

Pros and Cons of a public cloud

Pros

  • Scaleable
  • Pay for what you use
  • Easy Setup - hardware, application, and bandwidth costs are covered by the public cloud service provider
Cons
  • Customization may be limited
  • Potential security issues

Private Cloud

Private cloud infrastructure is a dedicated infrastructure provided to one single organization or client.

Pros and Cons of a private cloud

Pros

  • Performance
  • Control over data
  • Customization
  • Compliance
Cons
  • Expensive - Setup includes hardware, application, and bandwidth
  • Potential security issues - Private clouds are driven by concerns around security and compliance, and are required to keep assets within a firewall.
  • Private clouds are usually not an option for smaller to medium sized businesses, and it is typically put to use by larger corporation.
