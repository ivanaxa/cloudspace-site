Types of Clouds

Private Cloud

Private cloud is very expensive. This is normal not an option for Small to Medium sized business, and it is typically put to use by large corporation. Private cloud are driven by concerns around security and compliance, and require to keeping assets within the firewall.

Public Cloud

Public cloud is a multi-tenancy cloud environment which have multiple user share the same resource, but it is separate by multiple security layers from storage to user interface.
